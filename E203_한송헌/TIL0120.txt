프로젝트 게임 테이블 구상

1. 어항 상태 테이블 (AquariumState)
이 테이블은 어항의 현재 상태 정보를 저장합니다.

필드명	데이터 타입	설명
aquarium_id	INTEGER	어항 고유 ID (PK)
temperature	FLOAT	현재 온도 (°C)
ph_level	FLOAT	현재 pH 수준
lighting_status	BOOLEAN	조명 상태 (ON/OFF)
brightness	INTEGER	조명 밝기 (0~100)
water_level	FLOAT	현재 물 수준 (리터 단위)
filter_status	BOOLEAN	필터 작동 상태 (ON/OFF)
feeding_status	BOOLEAN	먹이 공급 상태 (ON/OFF)
last_update	DATETIME	마지막 상태 업데이트 시간

2. 사용자 테이블 (User)
사용자 정보 및 설정을 관리하는 테이블입니다.

필드명	데이터 타입	설명
user_id	INTEGER	사용자 고유 ID (PK)
username	VARCHAR(255)	사용자 이름
email	VARCHAR(255)	이메일 주소
password_hash	VARCHAR(255)	비밀번호 해시
preferred_theme	VARCHAR(50)	사용자 UI 테마 설정
created_at	DATETIME	사용자 가입 일자

3. 알림 테이블 (Notification)
어항의 상태 변화에 대한 알림을 관리합니다.

필드명	데이터 타입	설명
notification_id	INTEGER	알림 고유 ID (PK)
user_id	INTEGER	사용자 ID (FK)
message	TEXT	알림 메시지
notification_type	VARCHAR(50)	알림 종류 (온도 변화, 물 부족 등)
status	BOOLEAN	알림 읽음 여부 (읽음: TRUE, 안 읽음: FALSE)
created_at	DATETIME	알림 발생 시간

4. 먹이 공급 기록 테이블 (FeedingLog)
먹이 공급에 관련된 로그를 저장합니다.

필드명	데이터 타입	설명
feeding_log_id	INTEGER	먹이 공급 기록 고유 ID (PK)
aquarium_id	INTEGER	어항 고유 ID (FK)
feeding_time	DATETIME	먹이 공급 시간
amount	FLOAT	먹이의 양 (그램 단위)
status	VARCHAR(50)	먹이 공급 상태 (성공, 실패 등)

5. 어항 설정 테이블 (AquariumSettings)
어항의 각종 설정을 저장합니다.

필드명	데이터 타입	설명
setting_id	INTEGER	설정 고유 ID (PK)
aquarium_id	INTEGER	어항 고유 ID (FK)
temperature_set	FLOAT	설정된 온도 (°C)
feeding_timer	INTEGER	먹이 공급 타이머 설정 (초 단위)
filter_timer	INTEGER	필터 작동 주기 설정 (초 단위)
created_at	DATETIME	설정 생성 일시

6. 기기 연결 테이블 (DeviceConnection)
스마트 기기 연결 상태를 관리합니다.

필드명	데이터 타입	설명
device_id	INTEGER	기기 고유 ID (PK)
device_name	VARCHAR(255)	기기 이름 (예: 필터, 조명 등)
device_type	VARCHAR(50)	기기 유형 (필터, 조명, 히터 등)
aquarium_id	INTEGER	어항 고유 ID (FK)
connection_status	BOOLEAN	연결 상태 (ON/OFF)
last_connected	DATETIME	마지막 연결 시간

7. 어항 이벤트 로그 테이블 (AquariumEventLog)
어항의 주요 이벤트를 기록합니다.

필드명	데이터 타입	설명
event_id	INTEGER	이벤트 고유 ID (PK)
aquarium_id	INTEGER	어항 고유 ID (FK)
event_type	VARCHAR(100)	이벤트 종류 (온도 변화, 필터 작동 등)
event_details	TEXT	이벤트 상세 내용
event_time	DATETIME	이벤트 발생 시간
이 테이블 구조는 어항 관리 시스템의 주요 기능을 데이터베이스에서 효율적으로 처리할 수 있도록 설계되었습니다. 이 데이터를 기반으로 웹 페이지에서 실시간 모니터링, 제어 및 사용자 알림을 관리할 수 있습니다.
