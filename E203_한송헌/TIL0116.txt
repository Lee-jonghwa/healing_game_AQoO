{해야지} (가제)

컴퓨터를 주로 사용하고 마감기한이 중요한 개발자들을 위한 프로그램.
해야할 일을 세부 테스크로 나누어 설정 시간 안에 완료할 수 있도록 웹 푸시 알람으로 페이스를 도와주는 프로그램.

1. 로그인 / 회원가입 화면 
-> 유저 등록 정보, 아이디, 비밀번호

2. 메인 화면
- 최근 완료한 일들 목록 확인
- 할 일 등록 : 사용 언어, 프레임워크, 목적 (학습, 기능 구현), 할 일 명
- AI 할 일 바탕으로 세부 테스크와 소요 시간 추천하여 계획 제공
- (사용자가 계획을 자신에 맞게 수정)
- 할 일 시작
-> 최근 진행한 일들 목록 
-> 최종 결정된 일의 정보

3. 작업 화면
- 세부 테스크마다 소요 시간(남은 시간)과 완료 여부 확인
- 전체 할 일의 수행률 확인
- 세부 테스크 정보 수정
-> 수정된 계획 정보
-> 실시간으로 확인되는 웹 캠과 화면 정보

4. 푸시 알림
- 테스크의 마감 시간이 다가올 때
- 사용자의 자리 비움이 확인되었을 때
- 할 일 외의 다른 일을 하는 것 같을 때
- 휴식이 필요할 것 같을 때

5. 할 일  시간 완료 창
- 할 일의 최종 수행률 확인
- 총 할 일 소요 시간
- 각 세부 테스크의 소요 시간
- 전체 방해 받은 시간 
-> 최종 완료된 할 일의 정보

6. 마이페이지
- 사용자가 진행했었던 할 일 기록 확인
- 사용자의 수행 성공률 통계 확인
-> 사용자의 정보
